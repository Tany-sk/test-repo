DevTools

Виды сообщений:

  • Log Info - информационное сообщение. Отображается на белом фоне черным текстом.
  • Log Warning - предупреждающее сообщение. Говорит о том, что может возникнуть ошибка.
  • Log Error - сообщение об ошибке. Относим их разработчикам.
  • Log Table - сообщение в виде таблицы.
  • Log Group- группы значений.
  • Log Custom- кастомный лог, сообщение в произвольном формате.
  • Cause 404- вызывает ошибку 404, можем посмотреть, как она отображается в консоли.
